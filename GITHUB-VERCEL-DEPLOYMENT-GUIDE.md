## 📋 STEP-BY-STEP DEPLOYMENT

### STEP 1: GitHub Setup (5 minutes)
1. **Go to:** [github.com](https://github.com)
2. **Click:** "New repository" (green button)
3. **Create TWO repositories:**
   - `flipright-website` (for FlipRight site)
   - `viora-website` (for Viora site)
4. **Make them:** Public repositories
5. **Don't add** README, .gitignore, or license (we have files ready)

### STEP 2: Upload Files to GitHub (2 minutes each site)

**For FlipRight:**
1. **Open your new** `flipright-website` repository
2. **Click:** "uploading an existing file" 
3. **Drag and drop:** ALL files from `flipright-deploy/` folder
4. **Commit:** "Initial website upload"

**For Viora:**
1. **Open your new** `viora-website` repository  
2. **Click:** "uploading an existing file"
3. **Drag and drop:** ALL files from `viora-deploy/` folder
4. **Commit:** "Initial website upload"

### STEP 3: Vercel Deployment (3 minutes each site)
1. **Go to:** [vercel.com](https://vercel.com)
2. **Click:** "Start Deploying" or "New Project"
3. **Sign in with GitHub** (authorize access)
4. **Import** your repositories:
   - Import `flipright-website` → Gets URL like `flipright-website.vercel.app`
   - Import `viora-website` → Gets URL like `viora-website.vercel.app`
5. **Settings:** Use default settings (no changes needed)
6. **Deploy!** 🚀
