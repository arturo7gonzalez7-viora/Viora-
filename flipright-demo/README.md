## Customization

All styling is in the `<style>` section at the top of `index.html`.  

To change colors, edit the `:root` CSS variables:
```css
--primary: #dc2626;        /* Main red */
--primary-dark: #991b1b;   /* Darker red */
```

To update content, search for the section in the HTML and edit directly.

## Tech Stack

- Pure HTML5 + CSS3 (no frameworks)
- Google Fonts (Inter)
- Inline SVG graphics
- Vanilla JavaScript for smooth scrolling
